# Simple ERC-20 Token (Foundry)

This repository contains a streamlined, secure implementation of an ERC-20 token on Ethereum. It utilizes the industry-standard OpenZeppelin libraries to ensure safety and compatibility with major exchanges and wallets.

## Features
- **Standard ERC-20 Compliance**: Full support for transfers, allowances, and metadata.
- **Foundry Powered**: Blazing fast compilation and testing.
- **Flat Structure**: Simplified file layout for educational and rapid deployment purposes.

## Quick Start
1. Install Foundry.
2. Run `forge build` to compile the contracts.
3. Run `forge test` to execute the test suite.
